Welcome to studyBites!


This is our final project for Web Programming and Design (CMPS 278) - Spring 2020 at The American University of Beirut.

The banner pictures used accross the website are imported from pexels.com (free stock images). We did however alter the overlay color so that it suits the color palette of our choice.

For the Font-end, we used HTML/CSS, JavaScript, Bootstrap (for responsiveness) and Adobe Photoshop (for wireframes and images).
For the Back-end, we used PHP and MySQL.
